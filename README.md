Приложение для ведения личного списка фильмов. Позволяет добавлять записи, фильтровать их и автоматически сохраняет данные в JSON-файл.
Для создания фильмов и записи
Функции
- Добавление фильма (Название, Жанр, Год, Рейтинг).
- Валидация ввода (год — число, рейтинг 0-10).
- Фильтрация по жанру и году.
- Автоматическое сохранение в `movies.json`.

Примеры использования (Тесты)
1. Введите "Inception", "Sci-Fi", "2010", "8.8" -> Нажмите "Добавить".
2. Попробуйте ввести рейтинг "12" -> Получите предупреждение.
3. Введите в фильтр "Sci-Fi" -> В таблице останутся только соответствующие фильмы.
